## Что это
Лендинг-визитка психолога **Алины Дубовской** (EMDR / IFS / танцевально-двигательный терапевт).
Цель сайта — информационная (не продающая): чтобы пришедший по рекомендации мог прочитать про образование, подход, опыт.

## Стек и структура
- **Next.js 16 (App Router) + TypeScript + Tailwind v4**, статический экспорт (`output: export` → `out/`).
- Разделы (сверху вниз): Шапка → Обо мне → Образование → Принципы работы → О подходе → Консультация (с формой) → FAQ → Контакты.
- Ключевые файлы: `src/app/page.tsx`, `src/components/*`, контент `src/lib/defaultContent.ts` + хук `src/lib/useContent.ts`.
- Плавающая кнопка-конвертик (`FloatingContact`) с попап-формой; через 60с мерцает.
- На hero слева — блок цифр (15 000+ часов / 200+ клиентов / 10 лет), цвет золото `#a17c34`.

## Команды
- `bash sync.sh` — подтянуть последнюю версию (в начале работы на компе).
- `npm run dev` — локально на http://localhost:3000.
- `bash deploy.sh` — собрать и выложить статику на VPS.
- После правок — коммит и `git push` (тогда другой комп увидит актуальное).

## Админка
- Адрес: **alina.cd-agency.ru/admin**, вход логин+пароль (логин `alina`; пароль — в `config.json` на сервере / у Сергея, в репо не хранится).
- Сервис: `admin-server/` (Node/Express), на VPS в `/opt/alina-admin`, порт 3010, под **pm2** (`alina-admin`).
- nginx проксирует `/admin` и `/api` на 3010; `/content.json` и `/media/` отдаёт из `/var/www/alina-data` (alias).
- Данные (общие для обоих компов, живут на сервере): `/var/www/alina-data/` — `content.json` (тексты), `media/` (фото+PDF), `bookings.json` (заявки), `config.json` (учётка).
- Возможности: редактирование всех текстов, загрузка/замена фото и PDF, база заявок + CSV. Форма сайта шлёт в `/api/bookings`.
- ВАЖНО: структура дефолтов дублируется в `src/lib/defaultContent.ts` и `admin-server/public/app.js` — держать синхронно.

## Дизайн-направление (от Алины)
Тёплый бежево-коричневый минимализм: засечная типографика (Playfair Display), арки, волнистые разделители секций, текстуры, мотив «мозаика/калейдоскоп», лёгкий восточный акцент. БЕЗ анимаций, без кричащих цветов, воздушно, ADHD-friendly, mobile-first.
